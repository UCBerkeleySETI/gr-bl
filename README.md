# gr-bl
Breakthrough Listen GNURadio Module

Built on gnuradio 3.7.11

Guppi Raw source block for reading [Guppi raw format](https://github.com/UCBerkeleySETI/breakthrough/blob/master/doc/RAW-File-Format.md). 

To install:
 ```
 git clone 
 cd gr-bl
 mkdir build
 cd build
 cmake ..
 make
 make install
 ```
